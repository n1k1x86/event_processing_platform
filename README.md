# event_processing_platform

## Описание проекта

Event Processing Platform — это платформа для приема и асинхронной обработки событий из Kafka и через HTTP API.

Система принимает события, валидирует их, сохраняет, маршрутизирует на фоновые обработчики и выполняет пользовательские job handlers. Дополнительно платформа формирует аналитический контур для последующего анализа событий. В перспективе аналитика будет вынесена в отдельный сервис.

## Основные возможности

- ingestion событий через HTTP и Kafka;
- асинхронная обработка через фоновых воркеров;
- подключение пользовательских job handlers;
- хранение событий и статусов обработки;
- расширение аналитическим контуром;
- возможность дальнейшего выделения отдельных компонентов в сервисы.

## Цель проекта

Прокачать навыки проектирования event-driven систем: Kafka, фоновые job processors, хранение данных, retries, observability и эволюция архитектуры.
