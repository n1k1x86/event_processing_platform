# event_processing_platform

## Описание проекта

Event Processing Platform — это платформа для приема и асинхронной обработки событий  через HTTP API.

Система принимает события, валидирует их, сохраняет, маршрутизирует на фоновые обработчики и выполняет пользовательские job handlers.

## Основные возможности

- ingestion событий через HTTP;
- асинхронная обработка через фоновых воркеров;
- подключение пользовательских job handlers;
- хранение событий и статусов обработки;
- возможность дальнейшего выделения отдельных компонентов в сервисы.

## Цель проекта

Реализовать платформу для работы с фоновыми задачами
